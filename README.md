## Getting Started

1. Clone the repo
```bash
git clone https://github.com/uday-0408/Portfolio.git
```

2. Install dependencies
```bash
npm install
```

3. Run the app
```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in browser.
